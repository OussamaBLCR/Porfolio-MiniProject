Mes Projets Réalisés

Application Web de Gestion de Tâches

Description :


Développement d'une application responsive pour la gestion de tâches personnelles ou professionnelles. Les utilisateurs peuvent créer, modifier, supprimer et organiser des tâches en fonction de leurs priorités.


Technologies Utilisées :


  • Frontend : React.js, HTML, CSS
  • Backend : Node.js, Express.js
  • Base de Données : MongoDB

Fonctionnalités :


  • Authentification utilisateur
  • Interface utilisateur intuitive
  • Notifications par e-mail pour les tâches urgentes

Site E-Commerce avec Panier d'Achat

Description :


Conception et développement d'un site e-commerce complet avec un système de panier d'achat, de paiement en ligne et de gestion de produits.


Technologies Utilisées :


  • Frontend : HTML, CSS, JavaScript, Bootstrap
  • Backend : Django (Python)
  • Base de Données : PostgreSQL
  • Paiement : Intégration de Stripe

Fonctionnalités :


  • Catalogue de produits avec filtres de recherche
  • Panier d'achat dynamique
  • Paiement sécurisé en ligne

Jeu Vidéo en 2D avec Unity


Description :


Développement d'un jeu vidéo en 2D utilisant le moteur Unity. Le jeu est un platformer où le joueur doit résoudre des énigmes pour avancer dans les niveaux.


Technologies Utilisées :


  • Moteur de Jeu : Unity
  • Langage : C#
  • Graphismes : Adobe Photoshop

Fonctionnalités :


  • Contrôles fluides et réactifs
  • Design de niveaux variés
  • Système de score et de sauvegarde
